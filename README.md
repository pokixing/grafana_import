# Grafana Import
You can import a template dashboard into your grafana by using this program.
## Prerequisite:
* support python version 2.6/2.7
* Grafana needs to be installed. You can install it according to the [official documentation](http://docs.grafana.org/installation/rpm/)
* Grafana-server needs to run
## How to use:
* Invoke ./grafana with root permission to start import in guided mod
